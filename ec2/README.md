# EC2

- Provides scalable virtual servers (instances) in the cloud.
- Supports multiple instance types for different workloads.
- Allows custom AMIs (Amazon Machine Images) for rapid deployment.
- Offers flexible pricing: On-Demand, Reserved, Spot, and Savings Plans.
- Enables secure access via SSH, IAM roles, and security groups.
- Integrates with EBS for persistent storage.
- Supports auto-scaling and load balancing.
- Provides monitoring and management via CloudWatch.
- Allows placement in multiple regions and availability zones for high availability.
- Offers elastic IP addresses for static public IP assignment.
